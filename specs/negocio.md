[REQUERIMENTOS DE NEGOCIO]

Eu quero organizar minhas finanças com um agente de inteligência artificial utilizando o Whatsapp.

Funcionalidades iniciais:

- Cadastrar um Gasto/Investimento
- Alterar um Gasto/Investimento
- Excluir um Gasto/Investimento
- Consultar gastos, gerar resumos financeiros.

Gasto/Investimento:

- Nome
- Descricao
- Forma de pagamento: Cartao, PIX
- Nr de parcelas
- Categoria: Gastos Fixos, Transporte, Alimentação, Investimento, Lazer, Compras
- Valor

Caso tenha mais de uma parcela, eu preciso que mais de um registro seja incluído.

[REQUERIMENTOS TECNICOS]

Stack em Python com Langchain
Banco de dados em postgres
Código limpo e fácil entendimento
Sem comentários no código
Arquitetura de camadas.
Código direto sem excesso de validações
Evolution Api configurada para conexao no whatsapp
parametrizacao de numero via ENV
